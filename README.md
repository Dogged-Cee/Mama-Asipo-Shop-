# Mama Asipo Shop 🌐

Mama Asipo Shop is a web development and digital services business that helps small businesses build professional online presence and also sells products.

---

## 🚀 Project Overview

This is a simple modern business website built using:

- HTML
- CSS
- JavaScript

It includes:
- Business information
- Services section
- Website pricing packages
- WhatsApp order system
- Contact details
- Location (Ahero, Kisumu, Kenya)
- Google Maps integration
